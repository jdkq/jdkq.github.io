安装Vue-cli脚手架

我在使用deepin安装vue-cli时出现以下几个问题

  1. 在用过 npm install -g @vue-cli 时：出现
vue-cli安装错误1

起初我以为安装中途结束，因为一只猫突然跳上我的笔记本键盘
然后又重新输入一遍但还是报上述图所示的问题

  1. npm cache clean --force 清理缓存后决定再试一遍
vue-cli安装错误2

然后我就懵了，入坑中…
猜测是不是因为权限问题，以及npm版本问题考虑，果断的使用cnpm安装vue-cli

  1. sudo npm install cnpm -g --registry=https://registry.npm.taobao.org 之后再使用 sudo cnpm install -g @vue-cli来下载vue-cli脚手架
cnpm安装vue_cli脚手架
  1. 安装成功！开始上项目
